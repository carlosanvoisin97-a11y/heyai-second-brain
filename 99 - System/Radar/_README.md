# Radar — briefing per il cockpit AI OS

I file in questa cartella sono **briefing competitor/fornitori/mercato AI** generati dalla routine `radar-competitor` (`99 - System/Routines/radar-competitor.md`) e mostrati nel widget **Radar** del cockpit (`~/ai-os-cockpit`, repo separato, sola lettura).

- Un file per briefing: `YYYY-MM-DD - <Entità>.md`
- Frontmatter atteso: `title`, `date`, `summary` (1 frase), `source` (url)
- I file con prefisso `_` (come questo) **non** sono briefing: il cockpit li ignora.

Materiale **esterno/pubblico** — nessun dato cliente sensibile qui (§13).
